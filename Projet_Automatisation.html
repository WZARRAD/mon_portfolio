    Présentation du projet

    Présentation du projet:


    Entreprise cliente : Axeplane, spécialisée dans la fabrication de pièces pour l'aviation


    Projet : Mise en place d'un serveur Active Directory pour gérer le parc informatique


    Responsable du projet : Moi-même, technicien supérieur en système et réseaux, en collaboration avec Aaron, administrateur Système et Réseaux


    Communication du responsable technique :

    • Anthony (DSI d'AxePlane): Échange informel sur la proposition d'Aaron et votre idée d'un serveur Windows pour centraliser la gestion des nouveaux PC.

    • Aaron : envoi du document technique contenant la checklist pour la mise en place de l'Active Directory.

    Définition des besoins du client


    • Modernisation du parc informatique:

    • Passage de Linux Ubuntu 18.04 et 20.04 à de nouveaux postes clients sous Windows.

    • Homogénéisation du parc pour une gestion plus efficace.

    • Mise en place d'un serveur Active Directory:

    • Facilitation de l'authentification des utilisateurs.

    • Gestion centralisée des ressources.

    Planification en Sprints Agiles


    • Sprint 1 : Préparation du serveur Windows.

    • Sprint 2 : Installation et configuration des rôles serveur.

    • Sprint 3 : Préparation des postes clients.

    • Sprint 4 : Mise en place des stratégies de groupes.

    • Sprint 5 : Automatisation de tâches.

    Compétences évaluées


    • Installer et paramétrer un système d'exploitation serveur.

    • Activer les services réseaux du système d'exploitation des serveurs.

    • Développer les outils permettant d'automatiser les tâches de maintenance.

    • Gérer les droits d'accès aux données dans le respect des stratégies de sécurité.

    Planification

    Sprint 1 : Préparation du serveur Windows


    Outils utilisés: Hyper-V, ISO Windows Server 2022, Gestionnaire de disques, Paramètres réseaux.


    Etapes :


    • Installation de Windows Server 2022 sur une nouvelle VM avec 2 disques de stockage.

    • Initialiser le second disque de stockage avec la lettre E.

    • Configurer la carte réseau : IP fixe, masque, passerelle, et DNS.

    Sprint 2 : Installation et configuration des rôles serveur


    Outils utilisés: Server Manager, dcpromo (configuration AD DS), gestionnaire DHCP, WDS, Active Directory Users and Computers, Active Directory Administrative Center, File and Storage Services.


    Etapes :


    • Installer les services AD DS et DNS

    • Créer un nouveau domaine dans une forêt (axeplane.loc).

    • Vérifier l'accès à Internet et effectuer les mises à jours sur le serveur.

    • Installer les services DHCP, WDS, d’impression, et l’outil MDT

    • Créer une étendue DHCP avec les options nécessaires.

    • Configurer WDS comme serveur intégré à Active Directory et ajouter une image d’installation.

    • Créer les unités d’organisation et groupes de sécurité dans l’AD.

    • Créer des comptes utilisateurs et les affecter à leurs groupes respectifs.

    • Créer et configurer les accès aux dossiers communs partagés des différents services.

    • Installer une imprimante virtuelle pour tester le serveur d'impression partagé.

    Sprint 3 : Préparation des postes clients


    Outil utilisé : ISO Windows 10 Entreprise


    Etapes :


    • Installer Windows 10 Entreprise sur deux nouvelles machines représentant le PC technique et le PC du stagiaire.

    • Vérifier la configuration réseau obtenue via DHCP.

    • Vérifier l'accès à Internet et effectuer les mises à jour.

    • Joindre les machines au domaine de l'entreprise (axeplane.loc).

    • Connecter le compte utilisateur et celui du stagiaire pour première utilisation.

    Sprint 4 : Mise en place des stratégies de groupes


    Outils utilisés : GPMC (Group Policy Management Console), RSOP (Resultant Set of Policy).


    Etapes :


    • Créer une GPO interdisant l'utilisation de clés USB exclusivement pour les stagiaires.

    • Elaborer une GPO interdisant l'installation de logiciels sur les postes de travail des stagiaires.

    • Concevoir une GPO permettant l'accès automatique à l'imprimante PDF pour l'ensemble des employés.

    • Développer une GPO de mappage de lecteur pour connecter automatiquement les partages de services.

    • Tester l'application de ces GPO sur les ordinateurs et utilisateurs.

    Sprint 5 : Automatisation des tâches


    Outils utilisés : Powershell ISE, Active Directory Module, Gestionnaire de tâches Windows.


    Etapes:


    • Création d'un script de génération en masse d'utilisateurs dans l'Active Directory

    • Conditions : utilisation d'un fichier .csv, créer avec les utilisateurs leurs dossiers partagés personnels.

    • Vérifier l'application correcte des GPO et l'accès aux dossiers personnels et de service.

    • Elaboration d'un script de sauvegardes de données utilisateurs de (C:\Users) vers (E:\Sauvegardes).

    • Planifier l'exécution via le Gestionnaire de tâches et vérifier son bon fonctionnement.
